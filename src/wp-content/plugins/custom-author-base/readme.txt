Allows you to set a custom permalink base for your author pages.

== Description ==

You can change your author base permalink to anything you want like 'members' or 'users' instead of 'author'.


== Installation ==

Upload the Custom Author Base plugin to your 'wp-content/plugins' directory, activate it and then choose your new base author permalink in the WordPress Administration Panel under 'Settings->Author Base'.

That's it!
